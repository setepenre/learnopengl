# learnopengl

Repo following along https://learnopengl.com OpenGL tutorial

## prerequisites

- OpenGL 4.6.0
- GLFW 3.3
- glm 0.9.9.8

## build

```bash
$ make
```

## run

```bash
$ ./main
$ ./main 1920 1080
```

`main` takes 2 command line argument: window starting width and height ; default to 1280x720 if invalid values provided.
